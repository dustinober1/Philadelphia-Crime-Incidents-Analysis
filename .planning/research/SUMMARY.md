# SUMMARY.md

Key Findings (research synthesis)

Stack: Python 3.11+, pandas/geopandas, pyarrow, Prophet/ARIMA for forecasting, scikit-learn/XGBoost for classification, folium/contextily for mapping. Reproducible environment via `requirements.txt`/`environment.yml`.

Table stakes: Yearly trends, monthly seasonality, hotspot detection, per-offense filters, reproducible notebooks and exported reports.

Watch out for: timezone mismatches, missing geometry, population effects on choropleths, and weather-join challenges.
